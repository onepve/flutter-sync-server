CBox 云同步服务

CBox 跨设备数据同步后端 — 完整 API 参考（51 端点）

认证

所有需要认证的接口需要在请求头中携带 Authorization: Bearer <token>，token 通过登录获取 JWT。

未登录用户只能访问注册、登录、公共配置、密码重置等公开接口。
公开接口（无需认证）
端点方法说明
/healthGET健康检查
/api/auth/registerPOST用户注册（需邀请码）
/api/auth/loginPOST用户登录，返回 JWT token
/api/invite/publicGET获取公开邀请码
/api/auth/forgot-passwordPOST忘记密码，发送重置邮件
/api/auth/reset-passwordPOST重置密码（需 token）
数据同步（需认证）
端点方法说明
/api/sync/uploadPOST上传同步数据（AES 加密后）
/api/sync/downloadGET下载所有同步数据
/api/sync/statusGET获取同步状态摘要
/api/sync/deltaPOST增量同步（返回变更数据）
/api/sync/clearDELETE清空所有同步数据
个人资料（需认证）
端点方法说明
/api/auth/profileGET获取个人信息
/api/auth/profile/nicknamePUT修改昵称
/api/auth/profile/usernamePUT修改用户名（重签 token）
/api/auth/profile/emailPUT修改邮箱
/api/auth/profile/passwordPUT修改密码
/api/auth/profile/avatarPOST上传头像（FormData）
/api/auth/profile/avatar/imageGET获取头像图片（JWT）
TOTP 双因素（需认证）
端点方法说明
/api/auth/totp/setupPOST获取 TOTP 密钥和二维码
/api/auth/totp/verifyPOST验证并启用 TOTP
/api/auth/totp/disablePOST关闭 TOTP
账号管理（需认证）
端点方法说明
/api/auth/send-delete-codePOST发送注销验证码
/api/auth/verify-delete-codePOST验证注销码/TOTP
/api/auth/delete-accountPOST注销账号
